# Landing Page — Viki Diary

## Цель

Презентационная страница для бота. Формат: скинуть друзьям ссылку, чтобы они узнали о сервисе и могли перейти в бота. Не конверсионный лендинг, а wow-презентация продукта.

## Общие параметры

- **Язык:** русский (только)
- **Стиль:** тёплый, семейный — в тоне самого бота
- **Структура:** классический лонгрид с мягкими анимациями при скролле (fade-in + slide-up, 400–600ms, ease-out)
- **Адаптивность:** mobile-first (основная аудитория — Telegram-пользователи на телефонах)
- **Хедер:** нет фиксированного хедера/навигации — одностраничный скролл без меню
- **Тёмная тема:** не поддерживается (только светлая)
- **Аналитика:** не планируется

## Секции

### 1. Hero

- **Заголовок:** «Сохраняй моменты, которые не повторятся»
- **Подзаголовок:** «Бот для семейного дневника в Telegram»
- **CTA:** кнопка «Открыть в Telegram →» (ссылка на t.me/бот)
- **Фон:** градиент кремовый → персиковый
- **Анимация:** заголовок и кнопка — fade-in при загрузке
- **Визуал:** эмодзи 👶 над заголовком (крупный, 48–64px)
- **Hover на CTA:** затемнение фона на 10%, плавный transition 200ms

### 2. Проблема

Три боли, каждая с эмодзи-иконкой:

| Иконка | Заголовок | Описание |
|--------|-----------|----------|
| 📱 | Фото теряются в галерее | Тысячи снимков, а найти «первую кашу» — невозможно |
| 💬 | Чат — не дневник | Записи тонут среди сообщений и мемов |
| 🧠 | Воспоминания забываются | Через год не вспомнишь, когда было первое слово |

- **Анимация:** пункты появляются по очереди снизу (stagger)

### 3. Как это работает

4 шага в горизонтальной цепочке (на мобильных — вертикально):

1. 📸 **Отправь фото** (или голосовое)
2. 🤖 **Бот сохранит** (с датой и тегами)
3. 👨‍👩‍👧 **Семья увидит** (уведомление сразу)
4. ✨ **AI напишет** (саммари месяца)

Между шагами — стрелки `→` (на мобильных — `↓`).

- **Анимация:** шаги появляются слева направо с задержкой

### 4. Фичи

6 карточек в сетке 3×2 (tablet: 2×3, mobile: 1 колонка):

| Иконка | Название | Описание |
|--------|----------|----------|
| 📸 | Фото и видео | Отправляй медиа — бот сохранит в дневник |
| 🎤 | Голосовые → текст | AI расшифрует голосовое сообщение |
| 🏷️ | Авто-теги | Умная категоризация записей |
| 📊 | AI-саммари | Итоги месяца с анализом фотографий |
| 👨‍👩‍👧 | Совместный доступ | Пригласи партнёра или бабушку |
| 📱 | Мини-приложение | Красивый просмотр дневника прямо в Telegram |

- **Стиль:** белые карточки с тёплой рамкой (`#F0E4DC`), скругление 16px
- **Анимация:** scale-in (от маленьких к нормальному размеру)

### 5. Скриншоты

3 реальных скриншота в мокапах телефонов:

1. **Чат с ботом** — отправка записи (фото + текст, ответ бота «✅ Записано»)
2. **Лента мини-аппа** — карточки записей с фотографиями (центральный, крупнее)
3. **AI-саммари** — текст месячного саммари с эмодзи

- **Расположение:** центральный мокап крупнее остальных (акцент)
- **Фон:** градиент персиковый → кремовый
- **Анимация:** лёгкий параллакс при скролле (offset ±20px)
- **Формат:** реальные скриншоты в мокапах iPhone (светлый безель, мягкая тень rgba(0,0,0,0.1))

### Технические требования

- Статическая HTML-страница (или генератор — на усмотрение)
- Адаптивная вёрстка (mobile-first)
- Анимации при скролле (Intersection Observer или CSS-only)
- Минимальный вес: без тяжёлых фреймворков
- SEO мета-теги:
  - `<title>`: «Viki Diary — семейный дневник малыша в Telegram»
  - `og:title`: «Viki Diary — сохраняй моменты, которые не повторятся»
  - `og:description`: «Бот для семейного дневника в Telegram. Фото, голосовые, AI-саммари.»
  - `og:image`: 1200×630px (превью для мессенджеров)
- Размещение: тот же домен `viki.deazmont.ru` (корень `/`)
- Анимации: stagger-задержка между элементами — 100ms, параллакс — ±20px offset
